## 📥 安装 / Installation

**禁用内置英文脚本**  
禁用内置的英文脚本

在你的 `mpv.conf` 文件中添加以下一行：  
在你的``mpv.conf``文件中添加以下行：

``` ini 
加载统计覆盖=否
```
