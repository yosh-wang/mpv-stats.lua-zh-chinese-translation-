## 📥 安装

### 1️⃣ 禁用内置英文脚本

在 `mpv.conf` 文件中添加以下一行：

```ini
load-stats-overlay=no
```

### 2️⃣ 安装本脚本

将 `stats.lua` 放入 mpv 的脚本目录：

- **Windows** — `%APPDATA%\mpv\scripts\`
- **macOS**   — `~/.config/mpv/scripts/`
- **Linux**   — `~/.config/mpv/scripts/`
